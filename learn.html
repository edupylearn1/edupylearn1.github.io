  • 1: GPIO Pins

    An awesome feature of your Raspberry Pi is it's GPIO Pins. These allows us to connect to the outside world. You can connect output devices like LED's and Buzzers to them using jumper wires. Also, you can add input devices like buttons to trigger things like when to turn the LED on using different programming languages. Each pin has its own number so we can tell the Pi what is connected to what. GPIO stands for General Purpose Input Output.

  • 2: Understanding LED's

    LED's are output devices that can light up and come in a range of colours and types. As you may have noticed an LED has long and a short leg. The long leg is positive, this is where power comes through. The short leg is called negative, this is where we ground the connection. We need a resistor to limit the amount of current going through it. There are different types of resistors but the common one for LED's is 330 Ohm.

  • 3: Understanding Buzzers

    Buzzers are output devices that can make a one tone noise. types. Like an LED, a buzzer has a long and a short leg. The long leg is positive, this is where power comes through. The short leg is called negative, this is where we ground the connection. Buzzers do not need a resistor as they are 3 volts.
