Activity

  • 1: Create a Course

    A steady hand game needs a course for you to go round. Follow the steps in our video below to help you to create and prepare your own. It works like a button. When the wand touches the course it completes the circuit like a button does.

  • 2: Wire it up

    Now you have created your steady hand game. It is now time to wire it up (if you already haven't) and create a simple circuit too. Follow the diagram on the left and click to enlarge it.

  • 3: Let's Code

    Now it is time to code. Once our Raspberry Pi is all booted up, Go to the menu by clicking on the Raspberry Pi logo in the top left hand corner. Navigate to Programming then click on Python 3. Then File and New File. Click file in that new window and click on Save As, save it as steadyhandgame.py. Follow the code on the right hand side, click to enlarge it.

  • 4: What to do next

    Congratulations, you have completed the activity. But, would could you do next to take your game further? Maybe add an interface using GUIZero or some neopixels using pixelzero? We would love to see what you have made, share it with us via support@edupython.co.uk or @edupython on twitter!
